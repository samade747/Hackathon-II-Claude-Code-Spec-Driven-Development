# Evolution of Todo — Phase I (In-memory Console App)

This repository contains the **Phase I** implementation plan and specs for the Evolution of Todo hackathon project.

All production code must be generated by AI tools (Claude Code, Gemini, etc.) using the specs in this repo.

## Structure

- `.specify/` — subagent definitions and project context.
- `constitution/` — project constitution (non-functional rules).
- `spec/` — phase-level specs and test cases.
- `tasks/` — atomic tasks for generation.
- `plan/` — implementation plans.
- `checklist/` — progress checklist.
- `history/` — ADRs and prompt history.
- `src/` — to be generated (Python code).
- `tests/` — to be generated (pytest test files).

## Getting Started

1. Create a virtualenv and install pytest:

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -U pip pytest
```

2. Open this project in Claude Code.

3. Use `CLAUDE.md` and `history/prompts/001-phase-1.md` as a guide to generate code.

4. Run tests:

```bash
pytest -q
```

Iterate by updating specs until all tests pass.
